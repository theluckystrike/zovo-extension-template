## Template Features to Consider

For each extension, consider adding:

1. **Keyboard Shortcuts** - Power user feature, boosts WAU
2. **Dark/Light Mode** - User preference
3. **Export/Import** - Data portability
4. **Onboarding** - First-time user experience
5. **Usage Stats** - Local only, helps engagement

---

## Naming Convention

Format: `[Keyword]: [Benefit] | Zovo`

- Primary keyword in first 36 characters
- Benefit-focused subtitle
- Zovo brand suffix

Examples:
- "Tab Manager: Save Memory & Organize Tabs | Zovo"
- "JSON Formatter: Viewer & Validator for Developers | Zovo"
- "Password Generator: Secure Random Passwords | Zovo"
